# SyntaxWear - Tênis e Sneakers Online

A **SyntaxWear** é uma interface de e-commerce moderna e responsiva dedicada à venda de tênis e sneakers. O projeto foca em uma experiência de usuário fluida, utilizando um design contemporâneo para destacar coleções urbanas e futuristas.

## 🚀 Funcionalidades

- **Design Responsivo**: Interface adaptável para dispositivos móveis, tablets e desktop.
- **Navegação Modular**: Menus organizados por categorias (Masculino, Feminino, Outlet).
- **Seção Hero de Impacto**: Destaque principal para a coleção "Krypton One" com chamadas para ação (CTAs).
- **Categorização Visual**: Seção de categorias (Casual, Esporte, Moderno, Futurista) com efeitos de hover.
- **Grid de Produtos**: Layout flexível para exibição de sneakers em destaque.
- **Rodapé Completo**: Newsletter, links de navegação e integração com redes sociais.
- **Copyright Dinâmico**: O ano no rodapé é atualizado automaticamente via JavaScript.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica para melhor SEO e acessibilidade.
- **CSS3**: Estilização modular e moderna.
  - **CSS Variables**: Gerenciamento centralizado de cores, fontes e espaçamentos.
  - **Flexbox & Grid Layout**: Sistemas de posicionamento robustos.
  - **Componentização**: Estilos divididos por componentes para facilitar a manutenção.
- **JavaScript (ES6+)**: Lógica para interações dinâmicas simples.

## 📁 Estrutura de Arquivos

```text
ecommerce-syntaxwear/
├── css/
│   ├── components/       # Estilos específicos de cada componente
│   ├── base.css          # Estilos base e tipografia
│   ├── layout.css        # Definição de containers e grid principal
│   ├── reset.css         # Normalização de estilos entre navegadores

├── images/
│   ├── banners/          # Imagens de fundo e hero
│   ├── icons/            # Ícones de interface (SVG)
│   ├── logo/             # Identidade visual da marca
│   └── products/         # Galeria de fotos dos produtos
├── js/
│   └── main.js           # Scripts de comportamento do site
├── index.html            # Arquivo principal (página inicial)
└── README.md             # Documentação do projeto
```

## 🔧 Como Visualizar

O projeto é inteiramente front-end e estático. Para executá-lo:

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/ecommerce-syntaxwear.git
   ```
2. Navegue até a pasta do projeto e abra o arquivo `index.html` em seu navegador de preferência.
3. *Dica:* Se estiver usando o VS Code, utilize a extensão **Live Server** para visualizar as alterações em tempo real.

## 🎨 Design e Estilo

O projeto utiliza uma estética "Cyber/Urban", com cores escuras contrastando com tons vibrantes (conforme definido em `variables.css`), transmitindo modernidade e tecnologia, alinhado ao público de sneakers exclusivos.
